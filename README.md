# К.Т.5 — Безопасность API в Node.js + Express

Учебный проект по дисциплине «Безопасность веб-приложений».

## Цель
Продемонстрировать типовые уязвимости API:
- раскрытие избыточных данных;
- небезопасное обновление данных;
- отсутствие фильтрации ответов.

## Реализовано
- уязвимая версия API;
- эксплуатация уязвимостей;
- защищённая версия API;
- фильтрация ответов;
- проверка прав доступа;
- ограничение разрешённых полей для обновления;
- валидация входных данных.
